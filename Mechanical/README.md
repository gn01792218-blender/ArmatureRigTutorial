# 製作機械生物rig
## 實作工作流
1.先製作Rig <br>
    (1)先完成某一組的Leg Bone，把末端Bone作為IK Bone <br>
    (2)調整每個Bone的Lock axes<br>
    (3)使用鏡像或複製，製作其他Leg<br>
2.套上Mesh<br>
    可以使用automatic weight 或是 一個一個綁定Bone
```javascript
//1.先製作Rig的好處是，可以讓Mesh完全符合Bone
//2.搭配使用吸附工具，用格線來對齊骨架
```
